Pandora-ma Commands Guide

Command prefix: '.'

Type a '.' before every command. For example: .ma_reg

The currency used by Pandora-ma is Dharkmi (DRM).

ma_reg

ma_reg

Registers a gambling account based on your discord username. This step is required before participating in any mahjong gaming.

ma_list

ma_list

Returns the current balance of all players. The names are listed by order of registration. Players registered earlier are higher up on the list.

ma_bal

ma_bal (username)

Returns the current balance of the specified user. Use the target's discord username e.g. jleungnotveteran

ma_load

ma_load (username1) (username2) (username3) (username4)

Loads the 4 players into a game so that Dharkmi gained/lost after every single round can be calculated. The order of the 4 players does not matter. All 4 users must be registered.

ma_trans

ma_trans (winner) (points) (loser)

Transfers the correct amount of Dharkmi based on points of the winning hand from the loser(s) to the winner. If the winner wins by a self-drawn tile, input "self" in the losers field. The 3 other players will be charged accordingly.

ma_add

ma_trans (username) (amount)

Adds a specified amount to a user's balance.

Note: You must be Pandora's father to use this command. For some reason she refuses to help everyone else
